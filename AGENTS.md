- This project has two main parts: Idaztian markdown editor framework, and Idatz, which is a editor that uses this framework. There are more demo examples in `examples` directory.
- The host machine does not have an up-to-date Node.js installation, so we need to run node related commands always using Docker.
- There is already a docker-compose.yml file under Idatz.  The service is listening on port `5174`. If you need it, ensure the service is running. Run any script or command inside this container with with `docker compose exec idatz-dev`.
